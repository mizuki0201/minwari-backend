## アプリについて

### アプリ概要

本アプリは、友達とグループを作り、その中で割り勘ができるアプリです。
グループを作り、そのメンバーで出かけたりした際の支払管理を行うことができます。
また、グループ内でイベントを複数作成できるため、たくさん遊んだりするメンバーでグループを作れば、「次回多めに払ってくれればいいよ〜」といったように、割り勘して支払いが発生した分を次回出かける際などに繰り越すこともできます。
